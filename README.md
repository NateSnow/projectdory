# 🐠 Project Dory — DanDân Digital

Play **DanDân** (aka Forgetful Fish), Magic: The Gathering's beloved shared-deck format, right in your browser.

## What is DanDân?

DanDân is a 2-player MTG format where both players share a single 80-card deck and graveyard. The only creature is **Dandân** — a 4/1 Fish that takes just 5 hits to deal lethal damage. The format is all about stack interaction, library manipulation, and incremental advantage.

Based on the official **Secret Lair DanDân Deck** decklist.

## Features

- 🐟 **vs CPU** — Challenge the AI fish
- 🤝 **Local 2P** — Pass & play with a friend *(coming soon)*
- 🌊 **Online** — Play against fans worldwide *(coming soon)*
- 🎴 Card images from [Scryfall](https://scryfall.com) (Secret Lair art)
- 📱 Responsive design for desktop and mobile
